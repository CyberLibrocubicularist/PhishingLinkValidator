# Phishing Link Validator

This Python script helps validate the format and accessibility of URLs to determine if a link might be suspicious or unsafe.

## Features
- Validates the URL format.
- Checks if the link is accessible using an HTTP GET request.
- Alerts if the link could be inactive or unsafe.

## How to Use
1. Clone this repository.
2. Run the script with Python:
   ```bash
   python3 phishing_validator.py
